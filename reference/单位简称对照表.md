# 单位简称对照表

## 使用说明

**本对照表用于生成公文时自动匹配标准简称。**

**简称使用优先级（从高到低）：**

1. **用户明确指定** - 用户在输入中明确指定简称（如"简称省XX局"）
2. **对照表匹配** - 在本对照表中查找单位全称对应的标准简称
3. **本地配置** - 查看IDENTITY.md中的默认单位信息（仅本地环境有效）

**重要：**

- 本对照表是skill通用文件，适用于所有部署环境
- IDENTITY.md是本地配置文件，只适用于本地环境
- 部署到线上环境时，应确保对照表包含所有需要的单位信息

## 简称规则

### 1. 省级单位

- **格式**: 省 + 核心业务 + 厅/局/委
- **示例**:
  - 广东省教育厅 → 省教育厅
  - 广东省科技厅 → 省科技厅
  - 省政务和数据局 → 省政务和数据局

### 2. 市级单位

- **格式**: 城市 + 政府/业务 + 局
- **示例**:
  - 深圳市政府 → 深圳市政府
  - 深圳市政数局 → 深圳市政数局

### 3. 中央单位

- **格式**: 保持原名或使用官方简称
- **示例**:
  - 海关总署广东分署 → 海关总署广东分署（不简称）

***

## 使用方法

在生成公文时:

1. 首先查看IDENTITY.md中的默认单位信息
2. 如果用户明确指定简称,使用用户指定的简称
3. 如果没有,在本对照表中查找
4. 如果对照表中没有,按照简称规则生成
5. 在文号、落款等位置使用简称
